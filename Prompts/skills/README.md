## 使用建议

### 新手入门
1. 先阅读 **01-资料搜集与验证** 了解如何找到可靠信息
2. 再阅读 **02-笔记结构设计** 规划笔记结构
3. 写作时参考 **04-技术内容写作**

### 遇到问题时
1. 图片问题 → **03-图片资源管理**
2. 链接失效 → **06-问题排查与修复**
3. 版本回滚 → **05-版本控制与协作**

## 核心原则概览

| 技能 | 核心原则 |
|------|----------|
| 资料搜集 | **可靠性 > 数量** |
| 笔记结构 | **渐进式深入** |
| 图片管理 | **本地化优先** |
| 技术写作 | **讲故事，不堆砌** |
| 版本控制 | **小步快跑，频繁提交** |
| 问题排查 | **先定位，再修复** |

## 持续改进

这些技能文档会随着实践不断更新。如果你有更好的方法或发现了问题，欢迎补充！
